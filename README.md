# Data-Analysis-of-Indian-Colleges

**Project Title: Data Analysis of Indian College Data using Numpy, Pandas, and Seaborn**

**Project Description:**

In this project, we will perform data analysis on a dataset containing information about Indian colleges. The dataset consists of several columns, including College Name, Placement, Stream, UG Fee, PG Fee, Academic, and Rating. We will use the Python libraries Numpy, Pandas, and Seaborn to explore, clean, visualize, and draw insights from the data.

Data Exploration:
